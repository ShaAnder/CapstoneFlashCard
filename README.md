<h2 class="header2">Description</h2>
<p class="text-content">
Third capstone project designed to showcase somewhat advanced TKinter and pandas usage, this is a small flashcard app where the user is given a timer to guess a french word, before the english equivilant is shown. The user can then opt to say if they knew the word or not. If they knew the word game continues as normal, if not the word is then added to a words to learn csv, via pandas, which will be loaded on next launch if it exists.
